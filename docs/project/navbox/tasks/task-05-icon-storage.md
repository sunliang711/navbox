# 任务 05：icon 存储与访问

## 目标

实现 icon 上传、校验、保存、访问和数据记录能力。

## 技术方案

1. icon 文件保存到 `./data/uploads/`。
2. 仅 admin 可上传 icon。
3. 限制 MIME、扩展名、文件大小。
4. 文件名使用内容 hash 或 UUID，避免路径穿越和覆盖。
5. 通过 `/uploads/:file` 提供只读访问。
6. `icons` 表记录文件元数据。

## 验收标准

1. admin 可以上传合法图片 icon。
2. 游客不能上传 icon。
3. 非图片、超大文件、非法路径文件被拒绝。
4. 上传成功后网站可引用 icon。
5. icon 文件可通过公开静态路径访问。

## 实现步骤

1. 定义上传配置。
2. 实现文件校验。
3. 实现文件保存和元数据记录。
4. 实现上传 Handler。
5. 配置静态文件路由。
6. 补充安全测试。
