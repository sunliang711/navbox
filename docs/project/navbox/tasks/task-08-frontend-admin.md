# 任务 08：admin 前端管理能力

## 目标

实现 admin 登录、网站管理、Tag 管理、批量操作、导入导出和密码修改。

## 技术方案

1. admin 登录后通过 Session Cookie 调用管理接口。
2. 管理入口仅 admin 登录后展示。
3. 编辑网站使用弹窗或抽屉。
4. Tag 管理支持颜色、图标、排序、默认 Tag。
5. 批量操作支持多选网站。
6. 导入导出使用文件上传和下载。

## 验收标准

1. admin 可以登录、退出和修改密码。
2. admin 可以新增、编辑、删除网站。
3. admin 可以新增、编辑、删除 Tag。
4. admin 可以上传 icon。
5. admin 可以批量添加 Tag、移除 Tag、删除网站。
6. admin 可以导入导出配置。
7. 游客无法看到 admin 管理入口。

## 实现步骤

1. 实现登录页或登录弹窗。
2. 实现 Session 状态检测。
3. 实现网站编辑表单。
4. 实现 Tag 管理界面。
5. 实现批量选择和操作栏。
6. 实现导入导出交互。
7. 实现密码修改界面。

## 前置依赖

任务 05、任务 07。

## 风险

1. 管理态和游客态 UI 需要清晰隔离。
2. 批量操作需要明确二次确认和结果反馈。
