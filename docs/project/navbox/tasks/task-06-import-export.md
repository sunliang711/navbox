# 任务 06：导入导出

## 目标

实现 admin 配置导入导出，支持 icon 文件随配置一起导出和导入。

## 技术方案

1. 导出格式使用 zip 包。
2. zip 内包含 `navbox.json` 和 `icons/` 目录。
3. `navbox.json` 包含网站、Tag、绑定关系、常用状态、排序值和 icon 引用。
4. 支持导出全部、指定网站、指定 Tag。
5. 导入时校验结构。
6. 导入冲突时跳过已有数据并返回报告。
7. 导入失败时整体回滚。

## 验收标准

1. admin 可以导出全部配置。
2. admin 可以按指定网站或指定 Tag 导出。
3. 导出包包含 icon 文件。
4. admin 可以导入配置。
5. 冲突项会跳过并返回报告。
6. 导入失败不会产生半成功数据。

## 实现步骤

1. 定义导入导出 JSON Schema。
2. 实现导出选择器。
3. 实现 zip 打包。
4. 实现 zip 解包和结构校验。
5. 实现冲突检测。
6. 实现事务导入。
7. 补充导入导出测试。

## 前置依赖

任务 05。

## 风险

1. 数据库事务无法直接覆盖文件操作，需要临时目录和提交阶段。
2. 冲突策略必须在报告中清晰表达。
